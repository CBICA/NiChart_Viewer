Description
-----------
Data from public IXI dataset used as an example of NiChart data visualization

Contents
--------
├── data
│   ├── IXI_demog.csv           : Age and Sex values for the IXI sample
│   └── IXI_DLMUSE.csv          : DLMUSE ROI values for the IXI sample
├── README.txt                  : This README file
└── run_nichart_viewer_IXI.sh   : Script to launch NiChart_Viewer with the IXI data

Usage
-----
- Run the script
- Use actions provided in different plugins to:
  - Show data tables
  - Correct ROI values by ICV
  - Merge ROI values with demographic data
  - View ROI data
  - View ROI data together with centile curves
